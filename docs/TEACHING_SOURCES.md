## How These Sources Are Used

The code currently turns these sources into four structured outputs:

1. `focus`
   - 名师和教研最先看的评分重点
2. `pitfalls`
   - 高频失误与典型扣分模式
3. `actions`
   - 本次最该先改的动作建议
4. `selfCheck`
   - 给学生的自我纠错问题
